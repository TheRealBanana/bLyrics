# bLyrics 2.0
A small lyrics/control application for Foobar2000 written in Python using PyQt4 and the httpcontrol plugin for Foobar2000.

Requirements (most recent versions):<br>
Foobar2000<br>
foo-httpcontrol<br>
ajquery template for httpcontrol<br>
